Sobre o Projeto


Este projeto tem como objetivo criar uma página web simples para aprimorar as habilidades em HTML, CSS e JavaScript, incluindo um botão que permite alterar o tema escolhido, localizado no rodapé das páginas.


Tecnologias utilizadas: HTML5, CSS3, JavaScript
